# Crosswalk: Conservation Evidence to CMP Actions

This directory provides a semantic bridge between Conservation Evidence (CE) actions and the CMP Conservation Actions Classification (v2.0).

## 1. Logic
The crosswalk is generated by `crosswalk_ce_cmp.py`. Conservation Evidence maps each action to one or more action types using the CMP Action types v1.1. The script uses the CMP-provided mapping from CMP Actions v1.1 to CMP Actions 2.0 for compatibility with the CMP Actions taxonomy used in this project. While mappings were provided for each Action type, the subactions do not map identically from 1.1 to 2.0. Further, Conservation Evidence used action number 7 as a catch all, rather than as "External Capacity Building" as would be expected from v1.1. Anything tagged as action type is mapped to the CMP Actions UNCLASSIFIED / OTHER category. 

## 2. Execution
To regenerate the suggested mappings:
```bash
uv run python mappings/ce_to_cmp/crosswalk_ce_cmp.py
```

## 3. Mapping Properties
- **Exact Matches:** Used when CE Action labels match CMP v2.0 categories exactly.
- **Related Matches:** Used when a CE Action is a specific instance of a broader CMP category.
- **Target:** `sources/cmp/actions/cmp_actions_2.0.ttl`
